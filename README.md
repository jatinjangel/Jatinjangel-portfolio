# 🚀 Run Locally


## Clone Repository

```bash
git clone https://github.com/jatinjangel/Jatinjangel-portfolio.git
```


## Enter Project Folder

```bash
cd Jatinjangel-portfolio
```


## Install Dependencies

```bash
npm install
```


## Start Development Server

```bash
npm run dev
```


Open browser:

```
http://localhost:5173
```
